# RU-PlumeScan

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GEE](https://img.shields.io/badge/Earth_Engine-Required-green.svg)](https://earthengine.google.com/)

**Configurable Detection Surface for TROPOMI/Sentinel-5P methane, NOвВВ, and SOвВВ anomalies over Western Siberia.**

## Overview

RU-PlumeScan is a regional, configurable workbench for detecting gas anomalies over Western Siberia using TROPOMI/Sentinel-5P satellite data in Google Earth Engine. Built around peer-reviewed methodologies (Schuit et al. 2023, Beirle et al. 2019, Fioletov et al. 2020), it provides:

- **Per-gas detection** for CHвВД (threshold-based, regional climatology), NOвВВ (flux divergence), SOвВВ (wind-rotated plume fitting)
- **Configurable parameters** with named Presets (default, schuit_eq, imeo_eq, sensitive, conservative, custom)
- **Reference Catalog Adapter (RCA)** for cross-source comparison with Schuit 2023, UNEP IMEO MARS, CAMS Hotspot Explorer
- **Multi-gas evidence aggregation** вАФ novel event-level matching for industrial source attribution
- **Forward-compatibility** to ML-augmented detection (v2)

### 7. BaseIngester abstract class (Python)

–§–∞–є–ї: `src/py/rca/base_ingester.py`

Per RNA ¬І5.1:

```python
from abc import ABC, abstractmethod
from typing import Optional, Tuple
import pandas as pd

class BaseIngester(ABC):
    """
    Abstract base class for Reference Catalog Ingesters.
    
    Each Ingester implements:
    - fetch(): get raw data from source
    - validate(): verify against declared stats
    - to_common_schema(): convert to Common Plume Schema
    - upload_to_gee(): create GEE Asset
    """
    
    SOURCE_NAME: str  # override
    DECLARED_STATS: dict  # override
    
    @abstractmethod
    def fetch(self) -> pd.DataFrame:
        """Fetch raw data from source. Returns DataFrame."""
        pass
    
    @abstractmethod
    def validate(self, raw: pd.DataFrame) -> dict:
        """
        Verify raw data against DECLARED_STATS.
        Returns dict with verification results.
        Raises ValidationError if discrepancy > 5%.
        """
        pass
    
    @abstractmethod
    def to_common_schema(self, raw: pd.DataFrame) -> pd.DataFrame:
        """Convert raw to Common Plume Schema DataFrame."""
        pass
    
    def ingest(self, asset_id: str) -> str:
        """Full pipeline: fetch вЖТ validate вЖТ convert вЖТ upload. Returns GEE Asset ID."""
        raw = self.fetch()
        validation = self.validate(raw)
        common = self.to_common_schema(raw)
        return self.upload_to_gee(common, asset_id)
    
    def upload_to_gee(self, common: pd.DataFrame, asset_id: str) -> str:
        """Upload Common Schema DataFrame as GEE FeatureCollection."""
        from .upload_to_gee import dataframe_to_gee_asset
        return dataframe_to_gee_asset(common, asset_id, source=self.SOURCE_NAME)


class ValidationError(Exception):
    """Raised when reference catalog ingestion fails validation against declared stats."""
    pass
```

### 10. GEE Asset folders initialization

–Я–Њ—Б–ї–µ –љ–∞—Б—В—А–Њ–є–Ї–Є local repo вАФ —Б–Њ–Ј–і–∞—В—М GEE Asset folders structure —З–µ—А–µ–Ј Python script. –§–∞–є–ї: `src/py/setup/init_gee_assets.py`

```python
"""
Initialize GEE Asset folder structure for RU-PlumeScan.

Creates empty folders:
- RuPlumeScan/
- RuPlumeScan/backgrounds/
- RuPlumeScan/backgrounds/CH4/
- RuPlumeScan/backgrounds/NO2/
- RuPlumeScan/backgrounds/SO2/
- RuPlumeScan/industrial/
- RuPlumeScan/catalog/
- RuPlumeScan/catalog/CH4/
- RuPlumeScan/catalog/NO2/
- RuPlumeScan/catalog/SO2/
- RuPlumeScan/refs/
- RuPlumeScan/comparisons/
- RuPlumeScan/presets/
- RuPlumeScan/presets/built_in/
- RuPlumeScan/presets/custom/
- RuPlumeScan/runs/
- RuPlumeScan/validation/
- RuPlumeScan/validation/synthetic_injection/
- RuPlumeScan/validation/regression/

Usage:
    python -m src.py.setup.init_gee_assets
"""
import ee

def main():
    ee.Initialize(project='nodal-thunder-481307-u1')
    
    folders = [
        'RuPlumeScan',
        'RuPlumeScan/backgrounds',
        'RuPlumeScan/backgrounds/CH4',
        'RuPlumeScan/backgrounds/NO2',
        'RuPlumeScan/backgrounds/SO2',
        'RuPlumeScan/industrial',
        'RuPlumeScan/catalog',
        'RuPlumeScan/catalog/CH4',
        'RuPlumeScan/catalog/NO2',
        'RuPlumeScan/catalog/SO2',
        'RuPlumeScan/refs',
        'RuPlumeScan/comparisons',
        'RuPlumeScan/presets',
        'RuPlumeScan/presets/built_in',
        'RuPlumeScan/presets/custom',
        'RuPlumeScan/runs',
        'RuPlumeScan/validation',
        'RuPlumeScan/validation/synthetic_injection',
        'RuPlumeScan/validation/regression',
    ]
    
    project_root = 'projects/nodal-thunder-481307-u1/assets'
    
    for folder in folders:
        full_path = f'{project_root}/{folder}'
        try:
            ee.data.createAsset({'type': 'Folder'}, full_path)
            print(f'Created: {full_path}')
        except ee.EEException as e:
            if 'already exists' in str(e).lower():
                print(f'Exists: {full_path}')
            else:
                raise

if __name__ == '__main__':
    main()
```
