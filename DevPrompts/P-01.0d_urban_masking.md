## Контекст

**P-01.0b обнаружила:** Tyumen, Surgut, Novokuznetsk masked в NO₂/SO₂ regional baseline через 30 km industrial buffer вокруг collocated TPP/GRES. Это **scope inflation** — мы exclude entire urban regions instead of just industrial sources.

**Architectural decision (researcher approved):** Option (b) — explicit urban polygon layer separated from industrial buffer.

**Logic:**
- Pixel в urban_mask **AND** industrial_buffer → masked (cities collocated с industrial)
- Pixel в urban_mask **only** (no industrial) → **kept** в baseline (urban background distinct from industrial point sources)
- Pixel в industrial_buffer **only** (no urban) → masked (rural industrial — current behavior)
- Pixel в neither → clean baseline (current behavior)

**Source:** GHS-SMOD JRC Global Human Settlement, GEE Asset `JRC/GHSL/P2023A/GHS_SMOD/2030`, 1km resolution, validated globally.

**Threshold:** classes ≥22 (urban cluster и выше). Catches Tyumen/Surgut/Novokuznetsk, не masks rural settlements.

**Apply ко всем 3 gases (CH₄/NO₂/SO₂)** для consistency. CH₄ urban contamination smaller но heating systems contribute.

**Rebuild scope:** все 3 regional baselines нужно rebuild. Reference baseline NOT affected.

### Шаг 1: Build urban_mask Asset

**File:** `src/py/setup/build_urban_mask.py`

```python
"""
P-01.0d: Build urban mask from GHS-SMOC threshold ≥22 (urban cluster and above).

Source: JRC Global Human Settlement, 2030 epoch, 1km resolution.
Asset: JRC/GHSL/P2023A/GHS_SMOD/2030
Classes:
  30 = Urban centre
  23 = Dense urban cluster
  22 = Semi-dense urban cluster
  21 = Suburban (NOT masked, kept в baseline)
  13/12/11 = Rural (NOT masked)
  10 = Water (NOT masked)

Output: RuPlumeScan/urban/urban_mask_smod22
- Band: urban_mask (1 = urban, 0 = non-urban)
- AOI: canonical (60, 50, 95, 75)
- Scale: 1000m (native GHS-SMOD)

Provenance: full canonical Provenance (TD-0024 pattern).
"""

import ee
from src.py.rca.provenance import compute_provenance

ee.Initialize(project='nodal-thunder-481307-u1')

AOI = ee.Geometry.Rectangle([60, 50, 95, 75])

config = {
    'source_asset': 'JRC/GHSL/P2023A/GHS_SMOD/2030',
    'urban_threshold_class': 22,  # ≥22 = urban cluster
    'aoi_canonical': '(60, 50, 95, 75)',
    'output_scale_m': 1000,
    'algorithm_version': '2.3',
    'rna_version': '1.2'
}

provenance = compute_provenance(
    config=config,
    config_id='default',
    period='ghs_smod_2030',
    algorithm_version='2.3',
    rna_version='1.2'
)

# Load GHS-SMOD
smod = ee.Image('JRC/GHSL/P2023A/GHS_SMOD/2030').select('smod_code')

# Threshold ≥22: urban cluster and above
urban = smod.gte(22).rename('urban_mask').clip(AOI)

# Export к Asset
asset_id = 'projects/nodal-thunder-481307-u1/assets/RuPlumeScan/urban/urban_mask_smod22'

task = ee.batch.Export.image.toAsset(
    image=urban,
    description='urban_mask_smod22',
    assetId=asset_id,
    region=AOI,
    scale=1000,
    maxPixels=1e10
)
task.start()

# Wait for completion
import time
while task.status()['state'] in ['READY', 'RUNNING']:
    time.sleep(30)
    print(f"State: {task.status()['state']}")

if task.status()['state'] == 'COMPLETED':
    # Set asset metadata с canonical provenance
    ee.data.setAssetProperties(asset_id, provenance.to_asset_properties())
    
    # Add urban_mask-specific metadata
    extra_props = {
        'mask_type': 'urban_smod_threshold_class_geq_22',
        'source_asset': config['source_asset'],
        'description': 'GHS-SMOD urban cluster mask (classes 22+) for Phase 1d urban masking integration'
    }
    ee.data.setAssetProperties(asset_id, extra_props)
    
    # Log SUCCEEDED
    from src.py.rca.provenance import write_provenance_log
    write_provenance_log(
        provenance,
        status='SUCCEEDED',
        gas='NA',  # not gas-specific
        period='ghs_smod_2030',
        asset_id=asset_id
    )
    
    print(f"Urban mask asset created: {asset_id}")
    print(f"Run ID: {provenance.run_id}")
else:
    print(f"FAILED: {task.status()}")
```

### Шаг 2 (CORRECTED): Update build_regional_climatology.py

```python
def build_regional_baseline_for_month(gas: str, year: int, month: int, config: dict) -> ee.Image:
    """
    Build regional baseline для one month с combined urban + industrial masking.
    
    P-01.0d change: mask BOTH industrial_buffer AND urban areas.
    Result: cleaner baseline (excludes urban contamination), reduced coverage в urban regions.
    """
    # Load masks
    industrial_buffered = ee.Image(config['industrial_mask_asset'])
    # industrial_buffered: 1 = clean (kept), 0 = industrial (masked)
    
    urban_mask = ee.Image(config.get('urban_mask_asset',
        'projects/nodal-thunder-481307-u1/assets/RuPlumeScan/urban/urban_mask_smod22'))
    # urban_mask: 1 = urban, 0 = non-urban
    
    # Resample urban_mask к analysis scale
    urban_at_analysis = urban_mask.reproject(crs='EPSG:4326', scale=7000)
    
    # Combined: keep ONLY pixels which are clean AND non-urban
    industrial_clean = industrial_buffered.eq(1)
    non_urban = urban_at_analysis.eq(0)
    final_clean_mask = industrial_clean.And(non_urban)
    
    # Apply к TROPOMI collection (rest of pipeline same as before)
    coll = (ee.ImageCollection(config['source_collection'])
        .select(get_required_bands(gas))
        .filter(...)  # date, AOI, etc.
        .map(apply_qa_filter)
        .map(lambda img: img.updateMask(final_clean_mask)))
    
    # Reduce к monthly composite (median)
    monthly = coll.median()
    return monthly
```
