### 3.9 Wind validation (v2.3.1, TD-0031)

**Wind data source:** ERA5/HOURLY collection (NOT ERA5_LAND/HOURLY which has only 10m).

**Primary wind level:** 850hPa (~1500m, top of typical PBL).

Rationale:
- Column XCH₄ measurement averages tropospheric column heavily weighted toward PBL для near-surface emissions
- 850hPa matches better than 10m for plume-axis alignment
- 10m wind: surface-only, не representative для column-integrated XCH₄

Wind bands extracted:
- `u_component_of_wind_850hPa`, `v_component_of_wind_850hPa` (primary)
- `u_component_of_wind_10m`, `v_component_of_wind_10m` (recorded для sensitivity analysis)
- `u_component_of_wind_100m`, `v_component_of_wind_100m` (recorded)

**Sampling protocol:**
1. Sample wind (u, v) at cluster centroid + ±3h window around orbit time
2. Mean wind direction via vector averaging (not directional averaging — prevents 359°→0° wrap)
3. Plume axis from cluster geometry (eigendecomposition of pixel coordinates covariance matrix)
4. Alignment check: |plume_axis_deg - wind_dir_deg| ≤ 30° (tighter than ±45° для near-surface, since 850hPa flow more coherent)

**Edge cases:**
- Wind speed < 2 m/s: skip alignment, set `wind_consistent=null` (insufficient data)
- ERA5 hourly nearest к TROPOMI overpass time

**Recorded в event metadata:**
- `wind_level_hPa`: 850 (default)
- `wind_u_850hPa`, `wind_v_850hPa`, `wind_speed`, `wind_dir_deg`
- `wind_alignment_score`: numeric 0-1
- `wind_consistent`: bool|null

**Algorithm version:** v2.3.1 (TD-0031 patch).
```

```python
def classify_event(cluster):
    # Priority 1: Reference zone auto-override
    if cluster.matched_inside_reference_zone:
        return 'diffuse_CH4'  # by definition non-industrial (federal law)
    
    # Priority 2: Wetland heuristic (3 of 4 conditions)
    wetland_conditions = [
        cluster.area_km2 > 1000,
        cluster.compactness_ratio < THRESHOLD_TBD,  # max_z / sqrt(area_km2)
        cluster.date_utc.month in [6, 7, 8, 9],  # JJAS peak
        cluster.nearest_source_distance_km > 100 or cluster.nearest_source_id is None
    ]
    if sum(wetland_conditions) >= 3:
        return 'diffuse_CH4'
    
    # Priority 3: Wind ambiguity check
    if cluster.wind_consistent is False:
        return 'wind_ambiguous'
    
    # Priority 4: Industrial classification
    if cluster.nearest_source_id is not None:
        return 'CH4_only'  # industrial CH₄ detection (no NO₂/SO₂ matching в Phase 2A)
    
    # Default
    return 'wind_ambiguous'  # no source, no wetland pattern, signal-only

4. Apply retroactively к full catalog
5. Document threshold value в Algorithm v2.3.3 patch

Aligns с §3.11 confidence scoring (`matched_inside_reference_zone=true → C_total *= 0.3`). Reference zones are protected federal lands — by definition no industrial activity. Any detection inside MUST be natural (wetland, geological seep, fires, etc.).

exports.computeZScore = function(orbit_image, reference_baseline, regional_baseline) {
    // Input: per-orbit XCH₄ image, dual baselines
    // Output: z-score image (single-band 'z')
    // 
    // Per Algorithm §3.5:
    //   z = (obs - baseline_primary) / sigma_primary
    //   primary = reference if pixel inside zone, else regional
    
    var ref_value = reference_baseline.select('ref_M' + monthFromOrbit(orbit_image));
    var ref_sigma = reference_baseline.select('ref_sigma_M' + monthFromOrbit(orbit_image));
    var reg_value = regional_baseline.select('median_M' + monthFromOrbit(orbit_image));
    var reg_sigma = regional_baseline.select('sigma_M' + monthFromOrbit(orbit_image));
    
    // Use reference where defined, else regional
    var primary_value = ref_value.unmask(reg_value);  // careful: DNA §2.1.4 forbids unmask(0) but here masking c regional
    var primary_sigma = ref_sigma.unmask(reg_sigma);
    
    var z = orbit_image.subtract(primary_value).divide(primary_sigma).rename('z');
    return z;
};

exports.applyThreeConditionMask = function(orbit_image, z_image, baseline_value, params) {
    // Per Algorithm §3.6 — 3-condition mask
    // params: {z_min, delta_min_ppb, relative_min_ppb, annulus_inner_km, annulus_outer_km}
    // Returns: binary mask (1 where ALL 3 conditions met)
    
    var delta = orbit_image.subtract(baseline_value);
    
    // Condition 1: z-score
    var z_test = z_image.gte(params.z_min);
    
    // Condition 2: absolute delta floor
    var delta_test = delta.gte(params.delta_min_ppb);
    
    // Condition 3: relative-to-annulus
    // Compute median over 50-150 km annulus around each pixel
    var annulus_kernel = ee.Kernel.circle({
        radius: params.annulus_outer_km * 1000,  // outer
        units: 'meters'
    }).subtract(
        ee.Kernel.circle({
            radius: params.annulus_inner_km * 1000,  // inner
            units: 'meters'
        })
    );
    // NOTE: DNA §2.1.5 forbids ee.Kernel arithmetic — use focal_median с two passes instead
    
    var annulus_median = delta.reduceNeighborhood({
        reducer: ee.Reducer.median(),
        kernel: annulus_kernel
    });
    
    var rel_test = delta.subtract(annulus_median).gte(params.relative_min_ppb);
    
    return z_test.and(delta_test).and(rel_test).rename('anomaly_mask');
};

exports.extractClusters = function(mask_image, min_cluster_px) {
    // Per Algorithm §3.7
    // connectedComponents 8-conn, min cluster size
    
    var clusters = mask_image.connectedComponents({
        connectedness: ee.Kernel.square(1),
        maxSize: 256
    });
    
    var cluster_areas = clusters.reduceConnectedComponents({
        reducer: ee.Reducer.count()
    });
    
    var significant = cluster_areas.gte(min_cluster_px);
    return clusters.updateMask(significant);
};

exports.computeClusterAttributes = function(cluster_image, orbit_image, baseline_value, z_image) {
    // Convert clusters к FeatureCollection с attributes
    // 
    // Returns: FC of {centroid_lon, centroid_lat, area_km2, n_pixels,

        reducer: ee.Reducer.max().combine(ee.Reducer.mean(), '', true),
        scale: 7000
    });
    
    return with_delta;
};

exports.validateWind = function(cluster_fc, era5_collection, params) {
    // Sample ERA5 850hPa wind at cluster centroids
    // params: {wind_level_hPa, alignment_threshold_deg, min_wind_speed_ms}
    
    return cluster_fc.map(function(feature) {
        var centroid = feature.geometry().centroid();
        var date = ee.Date(feature.get('date_utc'));
        
        // Sample ERA5 ±3h window around date
        var era5_window = era5_collection
            .filterDate(date.advance(-3, 'hour'), date.advance(3, 'hour'))
            .select(['u_component_of_wind_850hPa', 'v_component_of_wind_850hPa']);
        
        var mean_wind = era5_window.mean();
        var sample = mean_wind.reduceRegion({
            reducer: ee.Reducer.first(),
            geometry: centroid,
            scale: 9000  // ERA5 native resolution
        });
        
        var u = ee.Number(sample.get('u_component_of_wind_850hPa'));
        var v = ee.Number(sample.get('v_component_of_wind_850hPa'));
        var wind_dir = u.atan2(v).multiply(180 / Math.PI);  // degrees
        var wind_speed = u.hypot(v);
        
        var plume_axis = ee.Number(feature.get('plume_axis_deg'));
        var angle_diff = wind_dir.subtract(plume_axis).abs().min(
            ee.Number(180).subtract(wind_dir.subtract(plume_axis).abs())
        );
        
        var wind_consistent = ee.Algorithms.If(
            wind_speed.lt(params.min_wind_speed_ms),
            null,  // insufficient data
            angle_diff.lte(params.alignment_threshold_deg)
        );
        
        return feature.set({
            wind_u_850hPa: u,
            wind_v_850hPa: v,
            wind_speed: wind_speed,
            wind_dir_deg: wind_dir,
            wind_alignment_score: ee.Number(1).subtract(angle_diff.divide(180)),
            wind_consistent: wind_consistent,
            wind_level_hPa: 850
        });
    });
};

exports.attributeSource = function(cluster_fc, source_points, params) {
    // 50 km radius search + ranking
    // params: {search_radius_km, source_type_priorities}
    
    return cluster_fc.map(function(cluster) {
        var centroid = cluster.geometry().centroid();
        var nearby_sources = source_points.filterBounds(
            centroid.buffer(params.search_radius_km * 1000)
        );
        
        var ranked = nearby_sources.map(function(source) {
            var distance = centroid.distance(source.geometry()).divide(1000);  // km
            var source_type = source.get('source_type_category');
            var priority = ee.Dictionary(params.source_type_priorities)
                .get(source_type, 999);
            return source.set({
                distance_km: distance,
                priority: priority
            });
        }).sort('priority').sort('distance_km');
        
        var best = ranked.first();
        
        return cluster.set({
            nearest_source_id: best.get('source_id'),
            nearest_source_distance_km: best.get('distance_km'),
            nearest_source_type: best.get('source_type_category')
        });
    });
};

**Note on DNA §2.1.5 (no kernel arithmetic):**

Annulus median computation requires "donut" shape (outer disk minus inner disk). Direct ee.Kernel arithmetic forbidden. Use **two-pass approach:**

1. Compute median over outer disk (e.g., 150 km)
2. Compute median over inner disk (50 km)
3. Approximate annulus median ≈ outer_median (assumes inner area small relative к annulus area)

OR alternatively use focal_median + manual exclusion of inner pixels (more accurate but slower).

Document choice в Algorithm §3.6 implementation note.

def adjust_boundary_event(event, config):
    """Inflate consistency_tolerance based on proximity к boundary."""
    centroid_lat = ee.Number(event.geometry().centroid().coordinates().get(1))
    
    distances_km = [ee.Number(centroid_lat.subtract(b)).abs().multiply(111) for b in config['zone_boundary_lats']]

        near_boundary,
        ee.Number(30).add(35),  # base 30 ppb + max boundary step
        30
    )
    
    return event.set({
        'consistency_tolerance_ppb': inflated_tolerance,
        'near_zone_boundary': near_boundary

def classify_event(event_props):
    # Reference zone auto-override
    if event_props.get('matched_inside_reference_zone'):
        return 'diffuse_CH4'
    
    # Wetland heuristic (3 of 4)
    wetland_conditions = [
        event_props['area_km2'] > 1000,
        event_props['compactness_ratio'] < THRESHOLD_TBD,
        event_props['date_utc'].month in [6, 7, 8, 9],
        event_props['nearest_source_distance_km'] > 100 or event_props['nearest_source_id'] is None
    ]
    if sum(wetland_conditions) >= 3:
        return 'diffuse_CH4'
    
    # Wind ambiguity
    if event_props['wind_consistent'] is False:
        return 'wind_ambiguous'
    
    # Industrial
    if event_props['nearest_source_id']:
        return 'CH4_only'
    
    return 'wind_ambiguous'
