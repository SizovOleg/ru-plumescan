### Шаг 1: Synthetic divergence pipeline test

**Sanity check** перед running на real data.

**Implementation:** `src/py/setup/test_dual_baseline_pipeline.py`

Generate synthetic test image с known Δ patterns:

```python
import ee
ee.Initialize(project='nodal-thunder-481307-u1')

# 3 synthetic squares (1°×1°) в 3 latitude bands:
# - Pattern A: Δ=10 ppb at (54.0°N, 88.0°E) — Kuznetsky band
# - Pattern B: Δ=30 ppb at (61.0°N, 75.0°E) — Yugansky band
# - Pattern C: Δ=50 ppb at (65.0°N, 80.0°E) — Verkhne-Tazovsky band

reference = ee.Image('projects/nodal-thunder-481307-u1/assets/RuPlumeScan/baselines/reference_CH4_2019_2025_v1').select('ref_M07')

# Synthetic regional = reference + known Δ in 3 squares, else = reference
square_a = ee.Geometry.Rectangle([87.5, 53.5, 88.5, 54.5])
square_b = ee.Geometry.Rectangle([74.5, 60.5, 75.5, 61.5])
square_c = ee.Geometry.Rectangle([79.5, 64.5, 80.5, 65.5])

synthetic_regional = (
    reference
    .where(ee.Image.constant(0).clip(square_a), reference.add(10))
    .where(ee.Image.constant(0).clip(square_b), reference.add(30))
    .where(ee.Image.constant(0).clip(square_c), reference.add(50))
)

# Run dual baseline cross-check на synthetic
delta = synthetic_regional.subtract(reference)

# Verify Δ matches expected within numerical tolerance
sample_a = delta.reduceRegion(ee.Reducer.mean(), square_a, 7000).getInfo()
sample_b = delta.reduceRegion(ee.Reducer.mean(), square_b, 7000).getInfo()
sample_c = delta.reduceRegion(ee.Reducer.mean(), square_c, 7000).getInfo()

assert abs(sample_a['ref_M07'] - 10) < 0.01, f"Pattern A failed: {sample_a}"
assert abs(sample_b['ref_M07'] - 30) < 0.01, f"Pattern B failed: {sample_b}"
assert abs(sample_c['ref_M07'] - 50) < 0.01, f"Pattern C failed: {sample_c}"

print('Synthetic test PASSED — pipeline correct')
```

If FAIL → escalate, не proceed к real data analysis.

If PASS → proceed.
