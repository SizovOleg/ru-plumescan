### 2. GPPD subset (Russian Federation power plants)

**Источник:** GEE-доступный dataset `WRI/GPPD/power_plants` (Global Power Plant Database, World Resources Institute).

В Python:
```python
import ee
ee.Initialize()
gppd = ee.FeatureCollection('WRI/GPPD/power_plants')
russia_plants = gppd.filter(ee.Filter.eq('country_long', 'Russia'))
zapsib_aoi = ee.Geometry.Rectangle([60, 55, 90, 75])
zapsib_plants = russia_plants.filterBounds(zapsib_aoi)
```

**Output:** список plants с координатами, capacity, fuel type. Конвертировать в Common-aligned schema:
```json
{
  "source_id": "gppd_<gppd_idnr>",
  "source_type": "power_plant",
  "source_name": "<name>",
  "country": "RU",
  "fuel_primary": "<Coal|Gas|Oil|...>",
  "capacity_mw": <number>,
  "operator": "<owner>"
}
```

Сохранить как `data/industrial_sources/gppd_zapsib_subset.geojson`.

### 3. VIIRS night lights threshold (proxy для unknown industrial points)

**Источник:** GEE dataset `NOAA/VIIRS/DNB/MONTHLY_V1/VCMSLCFG` (monthly composite night lights).

Для Западной Сибири 2022-2024 (последние полные годы):
1. Compute median composite за 3 года
2. Apply threshold (radiance ≥ X nW/cm²/sr) для выделения bright industrial pixels
3. Convert raster bright pixels → polygons → centroids
4. Filter:
   - Distance from human settlements (через `MODIS/006/MCD12Q1` или OSM urban) — exclude city lights, keep only industrial bright spots
   - Brightness threshold: индустриальные факелы значительно ярче city lights (typical >50 nW/cm²/sr)

Threshold value — **calibrate visually** на known facilities (Уренгой, Бованенково — газовые факелы должны попасть в bright pixels).

**Output:** `data/industrial_sources/viirs_bright_proxy.geojson` с centroids bright pixels. Source attribution:
```json
{
  "source_id": "viirs_<index>",
  "source_type": "oil_gas",
  "source_subtype": "viirs_proxy_unverified",
  "country": "RU",
  "viirs_radiance_mean": <value>,
  "verification_status": "unverified"
}
```

Эти точки помечены как `unverified` — они proxy для discovery, не verified sources. В Plume Event attribution они работают как fallback если manual sources не покрыли.

### 5. Industrial proxy mask raster

Файл: `src/py/setup/build_industrial_mask.py`

После того как `source_points` загружен — построить **раster mask**:
1. Берём source_points FeatureCollection
2. Buffer каждую точку на 15 km (Algorithm §3.4.1 default `industrial_buffer_exclude_km` = 30 km, но для **mask** делаем ширее, до actual buffer apply on climatology)
3. Reduce to image: 1 = within any buffer, 0 = clean
4. Save as `RuPlumeScan/industrial/proxy_mask` Image
