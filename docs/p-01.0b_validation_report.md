### Methodological notes

**Yugansky centroid NaN на regional side (point 1):**
- Reference asset has valid value at (74.5°E, 60.5°N)
- Regional climatology has NaN — the centroid coord falls within the 30 km industrial-buffered exclusion zone (Surgut oil region has many small proxy points в 30 km radius).
- This is **by design**: regional climatology positively excludes anything inside 30 km of any industrial proxy, whereas reference baseline anchors to zapovednik polygons that exclude infrastructure by definition. The two methods sample slightly different physical geometries — that's the dual baseline contract.

**Norilsk Nadezhdinsky NaN на regional side (point 5):** expected, industrial proxy itself + 30 km buffer.

**"Bovanenkovo region" valid value (point 6):** label is misleading — actual Bovanenkovo gas field centroid is ≈ (68.4°E, 70.4°N), the sampled coord (70.5°E, 70.5°N) is ~80 km east, in less-instrumented Yamal area. Real Bovanenkovo would be masked.

**Reference baseline extrapolation flag (suspect):** points outside all named reference zones (Mid-Yamal, Norilsk, Bovanenkovo region) all return identical `ref_M07 = 1863.354` — same value as Verkhne-Tazovsky (point 2). This suggests the reference asset spatially extrapolates beyond zone polygons rather than masking them. Worth verifying separately:
- If intentional (e.g., nearest-zone fallback) — document в RNA.
- If unintentional — `reference_CH4_2019_2025_v1` may need .updateMask(zone_mask) re-export.

**Not blocking** для NO₂/SO₂ submission — only valid-pair Δs are used для tolerance check, и all 4 pass.
