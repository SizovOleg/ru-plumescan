### 5.4. Shared October peak — regional synoptic signal (potential publication contribution)

Все 3 zones (3 different biomes, 3 different latitudes) показывают peak в October:
- Yugansky M10 = 1892.05 (wetland 60.5°N)
- Verkhne-Tazovsky M10 = 1893.96 (permafrost 63.5°N)
- Kuznetsky Alatau M10 = 1872.22 (mountain 54.5°N)

Это **не zone-specific feature**, а **regional synoptic signal** affecting Western Siberia AOI broadly. Three biomes ranging от wetland до mountain forest, three latitudes spanning 9° — все share October peak. Это suggests common atmospheric column-level driver, не biotic emission cycle.

**Hypotheses (don't pick yet — investigation deferred):**

1. **Autumn PBL collapse + surface accumulation.** Surface temperature drop (часто резкий в конце октября континентальной Сибири) → stable boundary layer formation → near-surface CH₄ накапливается → column XCH₄ rises. Согласуется с lowland zones, но truthfully Yugansky baseline увеличивается на ~+18 ppb от M07 до M10.

2. **MODIS snow filter edge case (NDSI 20-40 partial coverage bias).** При rapid October snow accumulation MODIS snow mask может triggering нестабильно — partial-coverage pixels могут pass filter в одних retrievals и fail в other, biasing aggregate. Если check time-series видно ли accumulation steps when MODIS snow flag toggles, это указывает на filter artefact.

3. **Soil-atmosphere shoulder season exchange.** Autumn soil moisture saturated, freeze-up ebullition spikes from lakes (literature: Walter Anthony 2010, Sasakawa 2012 для Yamal). Может contribute, но вряд ли объясняет отсутствие peak в Кузнецкий Алатау mountain forest.

4. **Continental transport pattern shift.** Autumn jet stream rearrangement → meridional transport increases over Siberia → boundary CH4 import from China / Central Asia. Synoptic, all-zones consistent.

**Action:** документируется в KNOWN_TODOS как TD-0009 «Investigate October peak regional pattern across biomes». **Это potential additional scientific contribution для tool-paper или separate publication** — first systematic empirical observation of cross-biome synchronous October XCH₄ peak in Western Siberia from TROPOMI L3 reference baseline.

**Не блокер** для current P-01.0a closure. Reference baseline functional с этой October peak — это **measured signal**, не bug. Detection algorithm в Phase 2A применяет `delta_vs_reference` где baseline уже includes October peak — corrected automatically.

### 5.5. Data quality caveats

#### Kuznetsky Alatau retrieval count limitation

Kuznetsky Alatau monthly counts 60-140 (vs Yugansky ~5000-8000, Verkhne-Tazovsky ~10000-20000) — **two orders of magnitude меньше** lowland zones. Cause: mountain cloud cover + SWIR retrieval challenges over snow/aspect-variable surfaces.

**Implications:**

- Zone-aggregate baseline still computable (zone-mean over ~80-140 valid pixels per month adequate).
- Sigma estimates noisier (e.g., M01 sigma=2.78 ppb с count=18 likely artefact of low N — actually masking high natural variance в data).
- **Phase 2A detection в latitude band 53-57°N (where Kuznetsky Alatau is primary reference per Algorithm §11.3 Step 4) может show reduced sensitivity** — fewer valid TROPOMI scenes для both reference and detection comparison.

Tracked в `docs/KNOWN_TODOS.md` TD-0010.

#### Q-mid month gap caveat (M02/M05/M08/M11)

Phase A diagnostics consistently fail для Q-mid centred months (M02, M05, M08, M11) с `User memory limit exceeded` — deterministic pattern, не throttle (sleep(60) не помог). Root cause: при Q-mid centring filter window covers 3 fully-valid months × 6 history years × ~30 daily L3 mosaics ≈ 540 images, exceeds reduceRegion memory ceiling.

**Phase B Export batch task** также potentially affected (server-side compute) — **status TBD** when task completes.

**Implications:**

- Phase B Asset will likely have valid bands только для 8 of 12 months (если Q-mid fails server-side too) или 11 of 12 (если Q-mid succeeds в batch context but M12 polar night skip).
- Phase 2A pipeline в biologically active transition months (M05 Май, M08 Август, M11 Ноябрь) needs handle gracefully:
  - **Option A:** temporal interpolation from neighbouring valid months (e.g., M08 = mean(M07, M09))
  - **Option B:** skip detection runs in those months
  - **Option C:** investigate root cause + refactor `build_zone_baseline_single_month` (chunked compute per-year-per-month, aggregate в Python)

Tracked в `docs/KNOWN_TODOS.md` TD-0008 — **priority HIGH** (was deferred, escalated после P-01.0a confirms pattern не fix-able by simple sleep).

## 6. Implementation issues encountered

### 6.1. GEE memory throttling

4 of 12 monthly compute calls failed с `User memory limit exceeded` в pattern M02/M05/M08/M11 (every 3rd month). Pattern points к cumulative GEE user-quota throttling (not per-request). Mitigation: `time.sleep(60)` между monthly compute. Documented as OpenSpec MC-2026-04-28-B.

### 6.2. Wrong original sanity expectations

Original DevPrompt §3 expectations (Yugansky July 1900-1950 ppb, amplitude 30-80 ppb) overestimated TROPOMI column XCH₄ amplitudes by ~3×. Cause: projection from in-situ surface CH₄ flux measurements to column observations did not account for boundary layer dilution and atmospheric mixing in the column total. Lesson learned, documented as OpenSpec MC-2026-04-28-A.

### Interpretation

- **Summer match well (0.6 ppb diff)** — both zones показывают similar low-summer XCH₄ values around 1842 ppb.
- **Winter divergence (35 ppb)** — Кузнецкий Алатау 35 ppb выше зимой, Алтайский nearly flat seasonally. Это physical signature высокогорного Алтайского biome:
  - Алтайский elevation 2000+ m, тонкий boundary layer над высокогорьем.
  - Зимой column XCH₄ над high-mountain reflects free-tropospheric mixing rather than surface-trapped boundary layer (как в lowland Кузнецком Алатау).
  - Результат: flatter seasonal cycle, divergent от lowland reference.
- **Both negative seasonal_diff** (winter > summer) — для CH₄ это unusual (обычно summer dominates wetland emissions). Для Кузнецкого Алатау: mountain forest без wetlands → no summer source enhancement; winter accumulation в stable boundary layer dominates. Для Алтайского: high-mountain seasonality less pronounced.
