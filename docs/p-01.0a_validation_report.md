- **Architecture validated.** Reference-anchored approach delivers cleaner wetland signature (Yugansky concentrated wetland +20-30 ppb vs article zone-mean), confirming intended design behaviour.

## 6. Implementation issues encountered

### 6.1. GEE memory throttling

4 of 12 monthly compute calls failed с `User memory limit exceeded` в pattern M02/M05/M08/M11 (every 3rd month). Pattern points к cumulative GEE user-quota throttling (not per-request). Mitigation: `time.sleep(60)` между monthly compute. Documented as OpenSpec MC-2026-04-28-B.

### 6.2. Wrong original sanity expectations

Original DevPrompt §3 expectations (Yugansky July 1900-1950 ppb, amplitude 30-80 ppb) overestimated TROPOMI column XCH₄ amplitudes by ~3×. Cause: projection from in-situ surface CH₄ flux measurements to column observations did not account for boundary layer dilution and atmospheric mixing in the column total. Lesson learned, documented as OpenSpec MC-2026-04-28-A.

### Interpretation

- **Summer match well (0.6 ppb diff)** — both zones показывают similar low-summer XCH₄ values around 1842 ppb.
- **Winter divergence (35 ppb)** — Кузнецкий Алатау 35 ppb выше зимой, Алтайский nearly flat seasonally. Это physical signature высокогорного Алтайского biome:
  - Алтайский elevation 2000+ m, тонкий boundary layer над высокогорьем.
  - Зимой column XCH₄ над high-mountain reflects free-tropospheric mixing rather than surface-trapped boundary layer (как в lowland Кузнецком Алатау).
  - Результат: flatter seasonal cycle, divergent от lowland reference.
- **Both negative seasonal_diff** (winter > summer) — для CH₄ это unusual (обычно summer dominates wetland emissions). Для Кузнецкого Алатау: mountain forest без wetlands → no summer source enhancement; winter accumulation в stable boundary layer dominates. Для Алтайского: high-mountain seasonality less pronounced.
- **DNA §2.1 запрет 16 enforced correctly** — Алтайский excluded из production reference baseline. Architecture работает as designed.
