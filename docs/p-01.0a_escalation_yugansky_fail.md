## 3. Cross-zone comparison (sanity-context)

All 3 zones show the **same anomalous pattern** — peak in October, no clear summer maximum:

| Zone | Centroid | M07 (Jul) | M10 (Oct) | Pattern |
|---|---|---|---|---|

That all three zones — at three different latitudes, three different biomes (wetland / permafrost taiga / mountain) — show the same October-peaking, weak-summer pattern strongly suggests the artefact is **not zone-specific** but **methodology-** or **data-source-specific**.

## 4. Hypotheses (ranked by likelihood)

### H1 (high) — TROPOMI L3 v02.04 (Earth Engine) ≠ L2 v18_17 (literature)

DevPrompt §3 itself acknowledges this difference. Schuit / Reuter / Lauvaux published wetland CH4 enhancements (+50-100 ppb summer) all use L2 v18_17 reprocessed product. Operational L3 v02.04 in GEE may have:

- Different bias-correction model that flattens seasonal amplitude
- Stricter cloud / aerosol filtering that drops summer convective scenes
- Coarser spatial averaging that diffuses hot wetland pixels

**Test:** compare with published Reuter et al. 2021 seasonal cycles for Bovanenkovo/Vasyugan using L2 directly. If L2 also shows 17 ppb amplitude — methodology issue, not L3 specific.

### H2 (medium) — Zone-mean aggregation diffuses hot pixels

Yugansky is 6500 km² total / 2946 km² useable. Hot wetland pixels (Vasyugan bog interior) get averaged with cooler taiga pixels around bog edges. **Algorithm §11.5 expectation** of "1900-1950 ppb summer" may be valid only for small wetland patches, not for zone-mean.

**Test:** drop zone-mean aggregation, take per-pixel max within zone in July. If max-per-pixel reaches 1920 ppb → enhancement signal exists, just diluted by zone-mean.

### H3 (medium) — October peak is real synoptic / boundary layer signal, not wetland

October peak in **all 3 zones** (very different biomes) suggests synoptic effect:
- Boundary layer collapse → CH4 accumulates near surface
- TROPOMI sensitivity to boundary layer better at lower surface temperatures
- Possibly trapped CH4 release as freeze-up begins on lakes (literature: ebullition spikes)

This is real signal, but not the wetland summer signal expected by DevPrompt §3 sanity assertions. **The sanity assertions themselves may be wrong** — or applicable only to L2 product.

### H4 (low) — Polygon issue

Three different polygons all give the same pattern → **unlikely to be polygon-specific**. Yugansky polygon was OSM-derived (3.8% area diff vs documented), Verkhne-Tazovsky 0.3% diff, Kuznetsky Alatau 0.5% — all within R2 tolerance.

### H5 (very low) — Implementation bug in `build_zone_baseline_single_month`

The pipeline uses `Reducer.median()` per Algorithm §3.4.0 Step 3 (robust to outliers). Same code passed unit-test signature. **Unlikely**, but to rule out: cross-check single zone for single month against manual `getInfo()` calculation.
